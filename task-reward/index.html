Installation and Setup

When you download the Script from, you will get an archive file in .zip extension.
You need to unzip that archive. When done, you should see the Script folder.

Please, follow the steps below to install the Script.

Step 1 - Log into your hosting space (server) via Cpanel or FTP client.

Step 2 - In extracted archive folder, find New upload folder

Step 3 - Upload the Contents of New upload folder to your server .

Step 4 - After uploading the files you should go to www.your-domain.com/install/ to begin installation ( just add /install/ to the end of your url ). Then you will be shown the below Screen. If All Server Requirements are Supported, Click Next to Proceed.

Step 5 - Enter your Database Connection details and Click Next ( database tables are created automatically ).

Step 6 - Enter your Desired Admin Login Credentials and Click Install Script ( see screenshots below ).

Step 7 - If Script is installed, then you'll be shown the below Screenshot.

Step 8 - Installation is Completed, but before using it, you need to Remove/Rename the install folder.

Congratulation, you are now ready to use Pocket Admin Panel or Admin Dashboard !!

Read Next part of our documentation, to know the Coniguration Settings and AdNetworks settings before using the Application.

Updating Admin Panel

WARNING : This Step is not Needed for FRESH Installation

Download the new update from CodyHub and Extract the archive file, when done, you should see Update Folder (zip file name can be changed from version to version).

Please, follow the steps below to update the Script.

Make A Backup First

Step 1 - Make A Backup First ( Both Files & Database), useful when something goes wrong in updating process..

Step 2 - Log into your hosting space (server) via Cpanel or FTP client.

Step 3 - Upload the Contents of your required update folder to your server . (overwrite)

Step 4 - After uploading the files you should go to www.your-domain.com/update/ Then you will be shown the below Screen.

Step 5 - Click on the Proceed to Update Button to update the Script ( see screenshots below ).

Step 6 - That's it ... it is now Updated and you can delete the update folder (Recommended to Delete)
